Permet d'appliquer une réduction en masse à tous les produits d'une catégorie spécifique dans WooCommerce.

== Description ==

L'extension WooCommerce Réduction par Catégorie vous permet d'appliquer automatiquement une réduction à tous les produits d'une catégorie spécifique. Vous n'avez plus besoin de définir des réductions individuelles pour chaque produit.

= Caractéristiques =

* Application d'une réduction en masse à tous les produits d'une catégorie spécifique
* Définition libre du pourcentage de réduction
* Personnalisation du badge de réduction (couleur et texte)
* Affichage d'informations de réduction sur la page détaillée du produit
* Affichage des informations de réduction dans le panneau d'administration lors de l'édition d'un produit

= Comment ça marche ? =

1. Allez à la page WooCommerce > Réduction par Catégorie
2. Sélectionnez la catégorie à laquelle vous souhaitez appliquer la réduction
3. Définissez le pourcentage de réduction (20% par exemple)
4. Personnalisez le texte et la couleur du badge de réduction
5. Enregistrez les paramètres
6. La réduction sera automatiquement appliquée à tous les produits de la catégorie sélectionnée

== Installation ==

1. Téléchargez les fichiers de l'extension dans le répertoire `/wp-content/plugins/woocommerce-category-discount`
2. Activez l'extension via le menu 'Extensions' dans WordPress
3. Configurez les paramètres depuis la page "Réduction par Catégorie" dans le menu WooCommerce

== Foire Aux Questions ==

= Comment la réduction est-elle appliquée ? =

La réduction est appliquée sur le prix affiché des produits. Les prix originaux sont toujours visibles (barrés) pour montrer la réduction.

= Puis-je appliquer des réductions à plusieurs catégories ? =

Actuellement, l'extension ne permet d'appliquer des réductions qu'à une seule catégorie à la fois. Cette fonctionnalité pourrait être ajoutée dans les versions futures.

= Puis-je désactiver temporairement la réduction ? =

Oui, vous pouvez simplement décocher l'option "Activer la réduction" dans la page des paramètres pour désactiver temporairement la réduction.
